# Carl Angelo Patano — ePortfolio (ITC-C508 / Elective 4)

A static ePortfolio site. No build step needed — just HTML, CSS, and SVG assets.

## Files
- `index.html` — landing page (About, Skills, Projects, **Course Expectations**, Contact)
- `itc-c508.html` — ITC-C508 course records page
- `style.css` — shared styles (dark theme)
- `assets/` — placeholder images (replace with your own photos/screenshots)

## Deploy: GitHub + Vercel

1. Create a new **public** repository on GitHub (e.g., `eportfolio`).
2. Upload all files in this folder to the repo (drag-and-drop on github.com works:
   *Add file → Upload files*). Keep `index.html` at the root.
3. Go to [vercel.com](https://vercel.com) → sign in with GitHub → **Add New → Project**.
4. Import your `eportfolio` repo → Framework Preset: **Other** → click **Deploy**.
5. Vercel gives you a live URL like `https://eportfolio-yourname.vercel.app` — it is
   public by default. Submit this URL to the ITC-C508 submission folder.

## Before submitting — personalize it
- Replace `assets/avatar.svg` and `assets/about.svg` with your real photos
  (keep the same filenames, or update the `src` in the HTML).
- Fill in your real projects in the Projects section (`index.html`).
- Update the GitHub link in the Contact section.
- **Important:** Review the Course Expectations answers and rewrite them in your own
  words based on your two required readings — the drafts are a starting point only.
